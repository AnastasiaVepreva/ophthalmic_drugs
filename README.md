# Ophthalmic_drugs
The eye is a complex anatomical structure protected by natural barriers, making drug delivery to the desired portion of the eye challenging. Like the creation of any drug, the creation of ophthalmic drugs takes a lot of time and money. In order to reduce the costs of creating eye medications, computer methods, in particular machine learning methods, are successfully used.

The goal of this project is to create a model that predicts the properties of ophthalmic drugs, such as corneal permeability, eye irritation, the ability of the drug to bind to melanin, and others.
